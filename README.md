# 2022_Team_Project

### 2022년도 체육종합 데이터 분석ㆍ활용 경진대회

[ 주제 ] 부산광역시 장애 인구 예측을 통한 장애인 스포츠 시설 확충 및 보완

- 과제 요약 
: 장애 인구와 장애인의 스포츠시설 이용률 증가로 장애인 전용 스포츠시설의 부족 문제가 더욱 심화되고 있다.   
: 이러한 문제가 큰 부산광역시의 장애인 인구를 예측하여 장애인 스포츠시설 확충 및 보완을 제안한다.
